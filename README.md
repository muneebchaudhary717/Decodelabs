# 📝 To-Do List Application (Python)

## 🚀 Project Overview

This is a simple yet powerful **To-Do List Application** built using Python. The project focuses on core backend concepts such as data management, user interaction, and program logic.

It allows users to manage their daily tasks efficiently through a command-line interface.

---

## 🎯 Features

* ✅ Add new tasks
* 📋 View all tasks
* ✏️ Update existing tasks
* ❌ Delete tasks
* 🆔 Unique ID for each task
* ⚠️ Error handling for invalid input

---

## 🧠 Concepts Used

* Python Lists (used as a mini database)
* Dictionaries (representing individual records)
* Loops and Conditional Statements
* Functions for modular programming
* IPO Model (Input → Process → Output)

---

## 🛠️ Technologies Used
